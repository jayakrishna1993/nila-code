# Nila Code VS Code Extension

This is the Nila Code VS Code Extension. Its primary jobs are

1. Implement the IDE side of the Nila Code IDE protocol, allowing a Nila Code server to interact natively in an IDE. This happens in `src/continueIdeClient.ts`.
2. Open the Nila Code React app in a side panel. The React app's source code lives in the `gui` directory.

# How to run the extension

See [Environment Setup](../../CONTRIBUTING.md#environment-setup)

# How to run and debug tests

After following the setup in [Environment Setup](../../CONTRIBUTING.md#environment-setup) you can run the `Extension (VSCode)` launch configuration in VS Code.
